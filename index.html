What is a Linear Combination?

A linear combination is an expression constructed from a set of terms by multiplying each term by a constant and adding the results.

For vectors \( \vec{a}, \vec{b}, \ldots, \vec{n} \) and scalars \( c_1, c_2, \ldots, c_n \), the linear combination is:

\( c_1 \vec{a} + c_2 \vec{b} + \ldots + c_n \vec{n} \)

Practice Problems

Express the vector (5,7) as a linear combination of (1,2) and (3,1).

Set up the equation: \( c_1(1) + c_2(3) = 5 \) and \( c_1(2) + c_2(1) = 7 \)

  1. Set up the system: \( 1c_1 + 3c_2 = 5 \) and \( 2c_1 + 1c_2 = 7 \)
  2. Solve the first equation for \( c_1 \): \( c_1 = 5 - 3c_2 \)
  3. Substitute into second equation: \( 2(5-3c_2) + c_2 = 7 \rightarrow 10-6c_2 + c_2 = 7 \rightarrow -5c_2 = -3 \rightarrow c_2 = 0.6 \)
  4. Then \( c_1 = 5 - 3(0.6) = 3.2 \)
  5. Check: \( 3.2(1,2) + 0.6(3,1) = (5,7) \)

Can the vector (4,6) be expressed as a linear combination of (1,1.5) and (2,3)?

Notice that (2,3) is exactly \( 2 \times (1,1.5) \). The vectors are linearly dependent, but (4,6) lies on the same line.

Key Points to Remember

  • A vector is a linear combination of others if it can be written using them with scalar coefficients
  • Not all vectors can be expressed as linear combinations of given vectors
  • The coefficients tell us "how much" of each vector to use
  • Linear combinations appear everywhere in math and science
